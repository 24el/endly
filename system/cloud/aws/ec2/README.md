# APIGateway EC2

This service is github.com/aws/aws-sdk-go/service/ec2.EC2 proxy 

To check all supported method run
```bash
    endly -s="aws/ec2"
```

To check method contract run endly -s="aws/apigateway" -a=methodName
```bash
    endly -s="aws/ec2" -a='stopInstances'
```
